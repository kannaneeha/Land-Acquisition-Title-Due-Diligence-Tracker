# Land Acquisition & Title Due Diligence Tracker

**Company:** Crownridge LLP  
**Document Type:** Internship Reference Document  

**Solution Name:** Land Acquisition & Title Due Diligence Tracker  

**What It Solves:**  
Manages land parcels under acquisition with survey numbers, title search status, encumbrance status, and legal clearance stages. 

**Why It Is Useful:**  
Reduces legal risk in land-intensive infrastructure projects by maintaining clear, centralized document trails instead of scattered manual records.

## Tech Stack

 Frontend
- HTML, CSS, JavaScript / React  
- Used for:
  - Land Acquisition & Title Due Diligence Entry Form  
  - Land Acquisition & Title Due Diligence Dashboard  
  - Status & Workflow Tracker  
  - Analytics charts and detail views 

 Backend
- Node.js (Express) **or** Python (Flask)  
- Used for:
  - `land_acquisition_title_due_diligenc` APIs
  - Workflow stage management engine
  - Report generation and data aggregation 

 Database
- MySQL **or** PostgreSQL  
- Tables:
  - `land_acquisition_title_due_diligenc`
  - `audit_logs`  
- With relationships, indexes, and audit logs for changes. 

 Charts / Visualisations
- Chart.js **or** Recharts  
- Used for:
  - Performance trend charts
  - Comparison analytics
  - Status distribution visualisations
